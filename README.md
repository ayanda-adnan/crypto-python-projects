# Crypto Python Projects

A collection of Python projects for working with cryptocurrency data.

## Projects
1. **[Bitcoin Price Tracker](bitcoin-price-tracker/)** – Fetches live Bitcoin price and logs to CSV.
2. **[Crypto Alert System](crypto-alert-system/)** – Monitors Bitcoin price and alerts when it drops below a threshold.
3. **[Portfolio Dashboard](portfolio-dashboard/)** – Calculates portfolio value and updates Google Sheets.

## How to Use
Each project folder contains its own README with setup instructions.

## Skills Demonstrated
- Python
- API integration (CoinGecko)
- Data handling (CSV, Google Sheets)
- Automation and error handling
- Version control (Git/GitHub)
