# Multi‑Tool AI Agent (v5)

An AI agent that can fetch price, get crypto news, and calculate RSI. Answers natural language questions like “Should I buy Bitcoin?” by combining live data.

## How to run
1. Install: `pip install langchain-groq requests feedparser talipp python-dotenv`
2. Set `GROQ_API_KEY` in `.env`
3. Run `python crypto_agent_v5.py`
