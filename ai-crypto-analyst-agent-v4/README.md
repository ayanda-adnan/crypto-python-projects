# AI Crypto Analyst Agent (v4)

Fetches live price, 24h change, 7‑day change from CoinGecko and produces structured analysis (short‑term trend, medium‑term trend, risk). Uses LangChain with Groq.

## How to run
1. Install dependencies: `pip install langchain-groq requests python-dotenv`
2. Set your `GROQ_API_KEY` in `.env`
3. Run `python crypto_agent_v4.py`
